# pkstack

pkstack is a published starter kit built around one idea:

your app code, your shared runtime, and your AI agent workflow should all agree on ownership from day one.

## What pkstack includes

- `create-pkstack` for web scaffolding
- `create-pkstack --mobile` for Expo scaffolding
- published shared packages under `@pkstack/*`
- strict shared config under `@pkstack/config`
- source-of-truth web and mobile templates inside the monorepo
- generated `AGENTS.md` and `CLAUDE.md` in scaffolded apps

## Current product state

- packages are published on npm at **v0.2.3** (verify the registry for the latest tag)
- templates and `create-pkstack` track the same semver in the monorepo
- docs are live at `pkstack.preetham.org`
