# Packages

## Runtime packages

### `@pkstack/ui`

Shared React components with typed variants exports for agent readability.

### `@pkstack/db`

Drizzle client creation, schema merging, and migration config helpers.

### `@pkstack/auth`

Better Auth factory helpers and the auth schema definition.

### `@pkstack/ai`

Vercel AI SDK wrappers and AgentKit helpers.

### `@pkstack/api`

Plain TypeScript contracts for shared API inputs and outputs.

These packages are versioned at `0.2.0` in the monorepo. Publishing them to npm is a separate release step that is still pending.

## Config package

### `@pkstack/config`

Shared tsconfig, ESLint, Tailwind, and the `require-variants` lint rule.
